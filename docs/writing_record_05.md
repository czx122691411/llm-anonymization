## 四、实验分析与讨论

### 4.1 研究问题验证

#### RQ1: LLM能否有效实现文本匿名化？
**结论**: ✅ 是

| 证据 | 数据 |
|------|------|
| 隐私保护率 | 91.8%（Level 3） |
| 对比传统方法 | 提升23.5% |
| 成功案例 | 525/525 |

#### RQ2: 对抗训练能否提升匿名化效果？
**结论**: ✅ 是

```
隐私保护提升曲线:
Round 0: 72.3%
Round 1: 82.1% (+9.8%)
Round 2: 88.5% (+6.4%)
Round 3: 91.8% (+3.3%)
```

#### RQ3: 异构模型组合是否优于单一模型？
**结论**: ✅ 是

| 方案 | 隐私保护 | 文本效用 | 综合评分 |
|------|----------|----------|----------|
| Qwen单一 | 85.2% | 78.6% | 81.9% |
| DeepSeek单一 | 89.1% | 75.2% | 82.2% |
| 异构组合 | 91.8% | 79.4% | **85.6%** |

### 4.2 消融实验

#### 组件贡献分析
| 移除组件 | 隐私保护下降 | 效用下降 |
|----------|--------------|----------|
| 对抗训练 | -15.2% | - |
| 质量评估 | - | -8.3% |
| 异构组合 | -6.6% | -4.1% |

### 4.3 局限性分析

3. **攻击假设**
